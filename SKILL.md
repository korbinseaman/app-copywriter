# APP文案生成器

帮助产品经理为手机系统应用编写专业文案，覆盖功能命名、Slogan、营销话术、交互提示和版本更新日志。

## 触发条件

当用户请求为手机系统应用编写文案时触发，包括：功能命名、Slogan、营销话术、交互提示、版本更新日志、文案调优。

不适用于：第三方 App 文案、纯技术文档。

## 使用流程

### 1. 收集输入

一次性确认以下信息（缺省时引导）：

- **应用名称**：哪个系统应用（"照片"、"备忘录"）
- **功能模块**：针对什么功能（"AI 修图"、"云同步"）
- **产出类型**：功能命名 / Slogan / 营销话术 / 交互提示 / 更新日志
- **目标平台**：Apple / Android / 全平台 / 不限（缺省：全平台）
- **侧重点**：想凸显什么（AI 能力、安全性、便捷性）
- **风格偏好**：科技感 / 亲和力 / 专业严谨 / 情感共鸣（缺省：科技感）
- **约束条件**：字数、版本数等限制

### 2. 加载参考

按产出类型读取对应文件：

- 功能命名 → `references/naming-conventions.md` + `references/terminology.md`
- Slogan / 营销话术 → `references/tone-guide.md` + `assets/examples/good-copies.md`
- 交互提示 → `references/naming-conventions.md` + `references/tone-guide.md`
- 更新日志 → `references/tone-guide.md`
- 同时参考 `references/case-library.md`

### 3. 生成文案

- 对齐目标平台风格
- 每次至少 3 个版本
- 输出格式精简：直接列出文案，附简短风格标签，不要表格、字数统计、依据说明

**输出格式示例：**

```
Slogan（≤ 12 字）

- AI 修图，一触即达
- 光影随心，AI 焕新
- AI 加持，随手出大片

功能介绍

能力驱动型

AI 修图 — 让 AI 成为你的私人摄影师。一键增强照片色彩、光线与细节……

效率驱动型

AI 修图 — 指尖轻触，瞬间调优色彩、光线和质感……
```

### 4. 质量检查

运行 `scripts/copy-validator.py <文案> [最大字数]` 检查，有问题时简要标注修正建议。

## 核心规范速查

- AI 功能用词：AI / 智能 / 智慧（按平台风格选择）
- 中英文混排加空格：「AI 修图」非「AI修图」
- 禁用词：史上最强、颠覆性、遥遥领先、革命性

# 版本更新日志模板

## 结构

```
[版本号] [发布日期]

【新增】
- [功能名]：[一句话描述 + 用户价值]

【优化】
- [功能名]：[优化了什么 + 体验提升]

【修复】
- [问题描述]：[已修复]

【已知问题】
- [问题描述]（可选）
```

## 写作原则

1. **分类清晰**：新增 / 优化 / 修复 三大类
2. **功能 + 价值**：不只说做了什么，要说给用户带来什么
3. **简洁条目式**：每条不超过 30 字
4. **避免技术语言**：用用户能理解的方式描述

## 示例

```
V3.2.0 2026-05-01

【新增】
- 智慧修图：AI 一键增强，让照片焕发专业质感
- 智能分类：按人物、场景、地点自动整理你的照片

【优化】
- 照片搜索速度提升 50%，搜索结果更精准
- 编辑工具新增 12 种滤镜和 5 种调色预设

【修复】
- 修复了部分 HEIC 格式照片无法正常显示的问题
- 修复了云端同步时偶发的重复文件问题
```

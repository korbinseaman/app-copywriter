# 应用商店描述模板

## 短描述（≤ 80 字，应用商店副标题）
- 结构：[核心卖点] + [差异化优势]
- 示例：「HarmonyOS 原生照片应用，AI 智慧修图让每一张照片都精彩」

## 长描述（300-500 字，应用商店详情页）

### 结构
```
【开场】一句话概括产品价值
【核心功能】3-5 个功能亮点，每个 1-2 句
【差异化】为什么选择我们
【行动号召】立即下载 / 体验
```

### 示例框架
```
[应用名] 是 HarmonyOS 原生 [应用类型]，为你带来 [核心价值]。

• [功能 1]：[一句话描述 + 用户价值]
• [功能 2]：[一句话描述 + 用户价值]
• [功能 3]：[一句话描述 + 用户价值]

依托 HarmonyOS [技术特性]，我们实现了 [差异化能力]。

立即体验，感受 [情感落点]。
```

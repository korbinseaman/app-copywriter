# 术语表

## 通用术语对照

| 概念 | 推荐用词 | 说明 |
|------|----------|------|
| 人工智能 | AI / 智能 / 智慧 | Apple 偏不强调 AI 概念，Google 强调 AI/品牌词，Huawei 用智慧 |
| 跨设备协同 | 同步 / 传输 / 流转 | Apple：接力/Handoff；Google：同步；Huawei：流转 |
| 云存储 | 云服务 / 云备份 | Apple：iCloud；Google：云端硬盘；Huawei：云空间 |
| 通知 | 通知 / 提醒 | 统一使用"通知" |
| 权限管理 | 权限管理 | 统一使用"权限管理" |
| 桌面小组件 | 小组件 / Widget | Apple：小组件；Android：Widget |

## 功能命名术语

| 类别 | 推荐用词 | 示例 |
|------|----------|------|
| AI 功能 | AI + 功能描述 / 智能 + 功能描述 | AI 修图、智能分类、智慧识物 |
| 图片处理 | 美化 / 编辑 / 精修 | 照片美化、智能编辑 |
| 分类整理 | 分类 / 整理 / 归集 | 智能分类、相册整理 |
| 搜索查找 | 搜索 / 查找 / 探索 | 照片搜索 |
| 分享传输 | 分享 / 发送 / 传输 | 一键分享、跨设备传输 |
| 存储备份 | 备份 / 云服务 / 保存 | 云端备份、本地保存 |
| 安全隐私 | 保护 / 隐私 / 加密 | 隐私保护、端到端加密 |

## 营销常用词汇

| 维度 | 推荐词汇 |
|------|----------|
| 速度 | 即刻、瞬时、极速、一触即达 |
| 智能 | AI 驱动、智能赋能、懂你所想 |
| 安全 | 安心、隐私守护、端到端加密 |
| 体验 | 沉浸式、无缝、自然流畅 |
| 生态 | 多端协同、无缝连接 |

## 中英文混排规范

- 英文单词前后各加一个空格：「AI 修图」而非「AI修图」
- 品牌名保留英文：iOS、Android、HarmonyOS、AppGallery、Google Photos
- 技术术语首次出现时中英并列：「小组件（Widget）」
- 后续出现时仅用中文
